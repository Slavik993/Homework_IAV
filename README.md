# Поиск по видео (аннотирование)

Программа ищет в видео моменты по **фотографии** или **текстовому запросу** (например «человек») и выдаёт результат в виде плеера с **гиперссылками на моменты** — по клику плеер переходит к нужному времени.

## Возможности

- Загрузка видео (MP4, AVI, MOV, MKV, WebM)
- Запрос: загрузка фото и/или ввод текста (промт)
- Модель CLIP (управляемая промтом сеть) для сравнения кадров с запросом
- Результат: встроенный плеер + список кликабельных таймкодов

При первом запуске поиска скачается модель CLIP (~600 MB).

## Запуск

```bash
python app.py
```

Откройте в браузере: **http://localhost:5000**

## Как пользоваться

1. Нажмите «Видео» и выберите файл — он загрузится на сервер.
2. По желанию загрузите **фото** (что искать в кадрах) и/или введите **текст** (например «человек», «собака», «красная машина»).
3. Нажмите **«Найти моменты»**. Поиск может занять время (извлечение кадров и работа CLIP).
4. Появятся плеер и список моментов. Клик по времени перематывает видео на этот момент.

## Технологии

- **Backend:** Flask, OpenCV (кадры), Transformers + CLIP (поиск по изображению и тексту)
- **Frontend:** HTML/CSS/JS, нативный `<video>` с перемоткой по клику
